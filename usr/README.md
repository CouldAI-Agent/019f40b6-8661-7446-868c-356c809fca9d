# Osmanlı Tarihi Tablosu

Bu uygulama, Osmanlı İmparatorluğu'nun kuruluşundan yıkılışına kadar olan dönemleri, padişahları, önemli savaşları, antlaşmaları, isyanları ve yüzyıllara göre ıslahatları bir tablo formatında sunan çapraz platform bir Flutter uygulamasıdır. 

## Özellikler
- **Dönemsel Görünüm:** Kuruluş, Yükselme, Duraklama, Gerileme ve Dağılma dönemlerini içerir.
- **Detaylı Bilgiler:** Tarih aralığı, padişah adı, önemli savaşlar, antlaşmalar, isyanlar ve yüzyıla göre ıslahatlar.
- **Responsive (Duyarlı) Tasarım:** Kaydırılabilir (scroll) tablo yapısı sayesinde hem mobil cihazlarda hem de daha büyük ekranlarda (Web/Masaüstü) sorunsuz görüntülenir.

## Teknolojiler
- Flutter
- Dart
- Material Design 3

## Kurulum ve Çalıştırma
Projeyi bilgisayarınıza klonladıktan sonra şu komutları çalıştırabilirsiniz:

```bash
flutter pub get
flutter run
```
